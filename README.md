# FirstRound

FirstRound automates technical pre-screening interviews using AI.
A recruiter creates a job post, generates interview questions, and shares a unique link with candidates.
The candidate uploads their CV and completes a fully vocal interview conducted by an AI in real time.
The recruiter then accesses a complete transcript, per-question scores, and a CV/job matching score for each candidate.

## How it works

**Recruiter side**

1. Paste a job description. HrFlow.ai parses it and extracts skills, location, and languages.
2. Review and validate the interview questions generated by the LLM.
3. Generate a candidate link and share it.
4. Access results: transcript, scores, and CV/job matching for each candidate.

**Candidate side**

1. Open the link and upload a CV. HrFlow.ai parses and indexes the profile.
2. Complete the vocal interview. The AI asks questions, listens, and follows up if an answer is too vague.
3. The session ends automatically. Results are saved and immediately visible to the recruiter.

**AI pipeline (real time)**

Audio from the candidate flows through a WebSocket connection into a sequential pipeline:

```
VAD (Silero) -> STT (Whisper) -> LLM (Ollama) -> TTS (Edge-TTS) -> audio back to candidate
```

- VAD detects end of speech from silence
- STT transcribes the audio segment to text
- LLM scores the answer (0-10), writes a short evaluation, and decides to move on or follow up
- TTS synthesizes the next question or follow-up and streams it back

## Stack

| Layer    | Technology                            |
|----------|---------------------------------------|
| Frontend | React, TypeScript, Vite, Tailwind CSS |
| Backend  | Python, FastAPI, WebSocket            |
| Parsing  | HrFlow.ai (jobs, profiles, indexing)  |
| LLM      | Ollama (llama3.2, local)              |
| STT      | Whisper (mlx-whisper)                 |
| VAD      | Silero                                |
| TTS      | Edge-TTS                              |

## Prerequisites

- Python 3.12+
- Node.js 18+
- [Ollama](https://ollama.com) installed and running locally
- A HrFlow.ai account with an API key, a board key, and a source key

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-org/firstround.git
cd firstround
```

### 2. Backend

```bash
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Create a `.env` file in the `backend` directory:

```env
API_KEY=your_hrflow_api_key
USER_EMAIL=your_hrflow_email
HRFLOW_BOARD_KEY=your_board_key
HRFLOW_SOURCE_KEY=your_source_key

OLLAMA_BASE_URL=http://localhost:11434
OLLAMA_MODEL=llama3.2

WHISPER_MODEL=mlx-community/whisper-large-v3-turbo
WHISPER_LANGUAGE=fr

TTS_VOICE=fr-FR-HenriNeural
FRONTEND_BASE_URL=http://localhost:5173
```

Pull the LLM model:

```bash
ollama pull llama3.2
```

Start the backend:

```bash
uvicorn app.main:app --reload
```

The API is available at `http://localhost:8000`.

### 3. Frontend

```bash
cd frontend
npm install
npm run dev
```

The app is available at `http://localhost:5173`.

## Environment variables

| Variable            | Description                             | Default                                |
|---------------------|-----------------------------------------|----------------------------------------|
| `API_KEY`           | HrFlow.ai API key                       | required                               |
| `USER_EMAIL`        | HrFlow.ai account email                 | required                               |
| `HRFLOW_BOARD_KEY`  | HrFlow board key for jobs               | required                               |
| `HRFLOW_SOURCE_KEY` | HrFlow source key for profiles          | required                               |
| `OLLAMA_BASE_URL`   | Ollama server URL                       | `http://localhost:11434`               |
| `OLLAMA_MODEL`      | Ollama model name                       | `llama3.2`                             |
| `WHISPER_MODEL`     | Whisper model identifier                | `mlx-community/whisper-large-v3-turbo` |
| `WHISPER_LANGUAGE`  | Transcription language                  | `fr`                                   |
| `TTS_VOICE`         | Edge-TTS voice name                     | `fr-FR-HenriNeural`                    |
| `FRONTEND_BASE_URL` | Frontend URL, used for candidate links  | `http://localhost:5173`                |
